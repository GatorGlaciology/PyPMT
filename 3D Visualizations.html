3D Visualizations¶

PyPMT offers 3D visualizations as well. For these visualizations, we will once again use bedmachine data.

In [6]:
nc = netCDF4.Dataset('C:/Users/blake/OneDrive/Desktop/BedMachineAntarctica-v3.nc')

x = nc.variables['x'][:]
y = nc.variables['y'][:]
bed = nc.variables['bed'][:]
surface = nc.variables['surface'][:]
thick = nc.variables['thickness'][:]
mask = nc.variables['mask'][:]

# reshape
bed2d = bed.reshape(13333, 13333)
surf2d = surface.reshape(13333, 13333)
thick2d = thick.reshape(13333, 13333)
mask2d = mask.reshape(13333, 13333)

# crop surface where ice thickness is 0
surf2d=surf2d.astype('float')
surf2d[thick2d <= 0] = np.NAN

Suppose we want to plot the bed elevation of Antarctica's Mount Vinson. To do so, we can call the following:

In [7]:
centerlat, centerlon = scar_loc('mount vinson')
plot_3ps(centerlat, centerlon, bed, color = 'r')

Additionally, if you call "%matplotlib notebook" before the previous code, you can rotate the visualization to view it from different angles.

We can also plot data in three dimensions using a colormap. In the example below, we display the bed elevation data for Thwaites Glacier. It is worth noting that our other bedmachine variables loaded above (surface, thick, mask) can also be plotted instead of bed elevation.

In [8]:
lat, lon = scar_loc('thwaites glacier')
surf_ps(lat, lon, x, y, bed, cmap = 'viridis')
